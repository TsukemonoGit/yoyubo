# yoyuboo

家計の「普通じゃない月」を記録するアプリ。

## デザイン思想

- 正確さよりも傾向の把握を重視
- シンプルな記録で家計の浮き沈みを可視化

## 主な機能

- 家計の支出・収入を万円単位で記録
- グラフで月別の変化を視覚化（未入力の月は直線補完して表示）
- バックアップ・リストア機能

## サポートプラットフォーム

| プラットフォーム | ストレージ | 状態 |
|-----------------|-----------|------|
| Android | Storage Access Framework (SAF) | 対応 |
| iOS | ファイル選択 | 対応予定 |
| Web | File System Access API + IndexedDB | 対応予定 |
| Desktop | - | 対応予定 |

## データの取り扱い

保存ファイルの場所は任意です。適当なフォルダに作って大丈夫ですが、Google Driveなどのクラウド共有フォルダ内にファイルを作成すると、複数端末でデータを同期できます。

## 技術スタック

- **フレームワーク**: Flutter
- **状態管理**: Provider パターン
- **データ保存**:
  - Android: SAF URIs
  - Web: File System Access API, IndexedDB, localStorage
  - バックアップ: shared_preferences

## データ仕様

- 金額単位は固定（万円）
- 小数点許容（例: 142.5）
- balance が null の月は「残高未入力」として扱い、グラフでは直線補完表示

## インストール

Android デバイス向けのビルドは別途準備中です。
